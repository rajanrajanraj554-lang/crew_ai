## ⚡ Installation (Day 1 Setup)

### Step 1: Install Python 3.10+
```bash
# Check your Python version
python --version

# Should be 3.10 or higher
```

### Step 2: Create Virtual Environment
```bash
cd /workspace/agentic_ai_master_course
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

### Step 3: Install Required Packages
```bash
# Core ML/AI packages
pip install numpy pandas scikit-learn matplotlib seaborn

# For later notebooks (LangChain, LangGraph)
pip install langchain langgraph langchain-community langchain-openai

# Vector databases
pip install chromadb pinecone-client weaviate-client

# Deep Learning
pip install torch transformers accelerate

# Development tools
pip install jupyterlab python-dotenv pydantic fastapi uvicorn

# Multi-agent frameworks (for later parts)
pip install crewai autogen

# Utilities
pip install requests beautifulsoup4 PyPDF2 python-docx
```

### Step 4: Launch Jupyter Lab
```bash
jupyter lab
```

## 💡 Tips for Success

### 1. Code Every Day
Even 30 minutes is better than nothing. Consistency beats intensity.

### 2. Don't Just Read - DO!
Type out the code yourself. Don't copy-paste. Muscle memory matters.

### 3. Break Things
Modify parameters, break the code, then fix it. That's how you learn.

### 4. Build Your Portfolio
After each notebook:
- Commit to GitHub
- Write a brief README
- Share on LinkedIn/Twitter

### 5. Join Communities
- LangChain Discord
- r/MachineLearning
- Local AI meetups
- Study groups

### 6. Teach Others
Write blog posts explaining what you learned. Teaching reinforces learning.

### 7. Don't Get Stuck
If something doesn't make sense:
- Move forward, come back later
- Ask in communities
- Check documentation
- Watch video tutorials on the topic

## 📞 Getting Help

### When You're Stuck:

1. **Check the error message carefully**
   - Read the full traceback
   - Google the specific error

2. **Review previous notebooks**
   - Concepts build on each other
   - You might need to refresh fundamentals

3. **Use these resources:**
   - LangChain docs: https://python.langchain.com
   - LangGraph docs: https://langchain-ai.github.io/langgraph
   - Hugging Face: https://huggingface.co/docs
   - Stack Overflow: Tag your questions

4. **Ask the community:**
   - Provide code snippet
   - Include error message
   - Explain what you're trying to do
   - Show what you've tried
